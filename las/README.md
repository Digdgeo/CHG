# Nubes de puntos LAS

Módulo de procesamiento y análisis de nubes de puntos LiDAR en formato LAS/LAZ.

## Clase 1 — Introducción a las nubes de puntos

### Contenidos
- ¿Qué es LiDAR? Principios de funcionamiento
- Formato LAS/LAZ: estructura y atributos
- Fuentes de datos: PNOA LiDAR (IGN), drones
- Visualización de nubes de puntos
- Clasificación de puntos (suelo, vegetación, edificios...)
- Generación de Modelos Digitales del Terreno (MDT) y Superficie (MDS)
- Análisis de vegetación: altura y cobertura

### Software y recursos
- **CloudCompare**: https://www.cloudcompare.org/
- **QGIS** con plugin LAStools
- **PNOA LiDAR (IGN)**: https://centrodedescargas.cnig.es/

### Datos de prácticas
Los datos utilizados en las prácticas se encuentran en la carpeta `datos/`.
