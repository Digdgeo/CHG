## Módulos

### Teledetección con SNAP y Sentinel
**Carpeta:** `teledeteccion/`

Introducción al análisis de imágenes de satélite con el software **SNAP** (Sentinel Application Platform) de la ESA. Se trabajará con imágenes de los satélites **Sentinel-2** (resolución media óptica) y **Sentinel-3** (océano y tierra).

### Nubes de puntos LAS
**Carpeta:** `las/`

Gestión, visualización y análisis de nubes de puntos en formato LAS/LAZ, con aplicaciones en generación de modelos digitales del terreno y análisis de vegetación.

### GeoPython
**Carpeta:** `geopython/`

Uso de Python para el análisis y procesamiento de datos geoespaciales. Librerías principales: `geopandas`, `rasterio`, `shapely`, `matplotlib` y otras del ecosistema geoespacial de Python.

### Google Earth Engine
**Carpeta:** `google_earth_engine/`

Análisis geoespacial a gran escala en la plataforma en la nube de Google Earth Engine (GEE), usando tanto la API de JavaScript como la API de Python (`geemap`).

---

## Requisitos

- SNAP (Sentinel Application Platform): https://step.esa.int/main/download/snap-download/
- Python 3.x con entorno geoespacial (ver `geopython/`)
- Cuenta de Google Earth Engine: https://earthengine.google.com/
